HTML

  • The head element contains information, or metadata, about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic HTML elements have names that describe the content/role they play on a webpage.
  • The !DOCTYPE declaration element tells the browser what document type it should expect when it is opened.
  • The html element represents the root, or start, of an HTML document.
  • An HTML attribute defines additional information about an element (always in opening tag).
  • The title element defines the title (name listed on tab) for a webpage and is important for SEO.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • Parent elements hold all the children elements.
  • Child elements are nested inside parent elements.
  • Div elements create a division in the page.
  • Span elements create an inline division.
  • H1-h6 elements represent the level of heading a given text block represents.
  • The p element represents are paragraph or block of text.
  • The ul element represents an unordered (bulleted) list.
  • The ol element represents an ordered list.
  • The li elements represent list items.
  • The img element contains information about images that are display on the webpage.
  • The src attribute defines the location, or source, of the image file.
  • The alt attribute contains a text string that describes the appearance and functionality of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The a element (anchor) creates links to the same webpage or other webpages.
  • The href attribute points to the URL for the link in the anchor element.
  • The main element holds the main content of the webpage.
  • The footer element appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.

Git

  • Git: Version control system that lets you manage and track each version of your project locally. Accessed via the command line.
  • GitHub: cloud-based hosting service that allows us to keep track of our projects inside repos that are stored remotely.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds all changes to file in working directory to the Git staging area.
  • git commit -m "message": commits staged changes to repo.
  • git pull origin main: pulls the base branch into your feature branch. Do this before pushing up any code.
